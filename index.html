Save this document in a GitHub repo, and rename it 'index.html'

go to 'Settings'

scroll down to 'Pages'

select 'Deploy from branch' and select the folder you'll be deploying from

you may not have multiple folders, but its important the index.html file and any corrresponding scripts, graphics, etc are all deployed from the same folder

check the 'Actions' tab to see the progress of your page, when it is green click on 'pages build and deployment'

voila! a link to your page will be provided
